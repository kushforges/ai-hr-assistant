# AI HR Assistant

A local AI-powered HR assistant built using:

- PostgreSQL
- Python
- Ollama
- Gemma LLM
- Faker

## Features

- Employee detail retrieval
- Attendance queries
- Top performer queries
- AI-generated HR responses
- Local LLM integration

## Technologies Used

- Python
- PostgreSQL
- Ollama
- Gemma
- Faker
- psycopg2

## Sample Queries

- What is the attendance of employee 45?
- Show employees with attendance below 70
- Top performing employees

## Setup

Install dependencies:

```bash
pip install faker psycopg2-binary ollama
```

Run:

```bash
python hr_assistant.py
```
